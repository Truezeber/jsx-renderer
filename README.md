# JSX renderer

JSX renderer made up by following [tutorial made by Jason Miller](https://jasonformat.com/wtf-is-jsx/)

## How to run

- Clone this repo:

```console
$ git clone https://github.com/Truezeber/jsx-renderer.git
```

- Install dependecies:

```console
$ npm install
```

- Start the project:

```console
$ npm run dev
```
